# VisionFlow

机器视觉流程编辑器 — 基于 HALCON + WPF (.NET 8)

## 功能

- **17个HALCON工具**：图像加载/阈值分割/形态学/Blob分析/定位器/查找线/查找圆/卡尺测量/距离计算等
- **蓝白主题UI**：分类工具箱（7大类），节点状态显示（红❌/绿✅/蓝⏳）
- **单步运行**：右键"单步运行到此处"，实时预览处理效果
- **HALCON图像预览**：结果显示窗口，显示图像/区域/数值
- **工具名称可修改**：双击编辑对话框顶部修改
- **流程保存/加载**：JSON格式 (.vflow)
- **参数实时刷新**：修改参数→应用→立即预览
- **压力测试**：连续运行999次
- **2小时稳定性测试**：StabilityTest.cs

## 环境要求

- .NET 8 SDK (Windows)
- HALCON 22.11 / 23.11 / 24.11（自动检测）

## 快速开始

```bash
cd VisionFlow
dotnet build
dotnet run
```

## 项目结构

```
VisionFlow/
├── Tools/          # 17个HALCON工具实现
├── Views/          # WPF窗口和对话框
├── ViewModels/     # MVVM ViewModel
├── Models/         # 数据模型
└── *.cs            # 核心组件（FlowRunner, HALCONPerfConfig等）
```
